### 任务

### 经验教训

**错误1：没有遵守技能优先原则**
- 直接用了底层 `browser` 工具
- 应该先读取 `Agent Browser` 的 SKILL.md

**错误2：中文输入问题**
- agent-browser 的 `fill`/`type` 命令不支持中文
- 解决方案：使用 `eval` 命令执行 JavaScript
- 示例：`agent-browser eval "document.querySelector('#kw').value = '中文'"`

**正确流程**
1. 识别任务类型 → 匹配技能
2. 读取 SKILL.md
3. 按照文档执行
4. 遇到问题查看 Troubleshooting
5. 使用替代方案（eval）
6. 最后才回退到底层工具
