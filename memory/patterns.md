# 工作模式

## 技能使用模式

### 技能优先原则（2026-03-08学到）
**场景**：遇到任务时
**错误做法**：直接使用底层工具
**正确做法**：
1. 识别任务类型
2. 匹配对应技能
3. 读取 SKILL.md
4. 按文档执行
5. 遇到问题查 Troubleshooting
6. 最后才回退到底层工具

**原因**：技能文档包含实践中积累的经验，能避免重复踩坑

### 中文输入处理（2026-03-08）
**问题**：agent-browser 的 `fill`/`type` 不支持中文
**解决方案**：使用 `eval` 执行 JavaScript
```bash
agent-browser eval "document.querySelector('#kw').value = '中文'"
```
