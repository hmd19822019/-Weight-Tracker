这是一个占位文件。
由于本地环境没有Java，无法直接生成密钥。

请按以下步骤操作：

1. 等待GitHub Actions构建完成（约5分钟）
2. 访问：https://github.com/hmd19822019/-Weight-Tracker/actions
3. 点击最新的workflow运行
4. 下载 "keystore-setup-info" artifact
5. 解压得到 keystore-base64.txt
6. 复制文件内容
7. 添加到GitHub Secrets:
   - 名称: KEYSTORE_BASE64
   - 值: 粘贴复制的内容

或者，如果你有Java环境，可以本地生成：

keytool -genkey -v -keystore weight-tracker.keystore \
  -alias weight-tracker \
  -keyalg RSA \
  -keysize 2048 \
  -validity 10000 \
  -storepass "WeightTracker2024" \
  -keypass "WeightTracker2024" \
  -dname "CN=Weight Tracker, OU=Development, O=Weight Tracker App, L=Shanghai, ST=Shanghai, C=CN"

然后转换为base64：
# Linux/Mac
base64 -w 0 weight-tracker.keystore

# Windows PowerShell
[Convert]::ToBase64String([IO.File]::ReadAllBytes("weight-tracker.keystore"))
