# 食物热量识别功能 - 实现方案

## 功能概述
在体重记录app中添加食物拍照识别热量功能

## 技术方案

### 方案选择：百度AI食物识别API
- **API**: 百度AI开放平台 - 菜品识别
- **免费额度**: 每天500次（足够个人使用）
- **识别内容**: 食物名称 + 热量 + 营养成分
- **准确率**: 85%+

### 实现步骤

#### 1. 注册百度AI账号
- 访问：https://ai.baidu.com
- 注册并创建应用
- 获取 API Key 和 Secret Key

#### 2. 添加拍照功能
使用 HTML5 Camera API（无需Cordova插件）
```html
<input type="file" accept="image/*" capture="camera">
```

#### 3. 调用识别API
```javascript
// 1. 获取Access Token
// 2. 上传图片（Base64）
// 3. 解析返回的食物信息和热量
```

#### 4. 显示结果
- 食物名称
- 热量（千卡）
- 营养成分（蛋白质、脂肪、碳水）
- 建议摄入量
